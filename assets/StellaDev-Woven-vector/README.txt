StellaDev — Prism (Woven / 엮인 프리즘) 벡터 로고
================================================

■ 포맷 안내
-----------
.ai는 모든 마크를 '채워진 다각형(fill)'으로 아웃라인해 제작했습니다.
이전 버전에서 일부 앱이 획(stroke)을 다르게 해석해 깨져 보이던 문제를
근본적으로 제거한 방식으로, 어느 뷰어·편집기에서도 동일하게 렌더됩니다.
SVG는 획(stroke) 기반이며 모양은 동일합니다 — 둘 다 원본으로 사용 가능합니다.

Illustrator: .ai 또는 .svg를 File > Open 으로 열면 편집 가능한 벡터로 열립니다.

파일
----
SVG (권장 · 모든 앱 호환)
• StellaDev-Woven-symbol.svg            심볼 (투톤: 잉크+인디고)
• StellaDev-Woven-symbol-mono.svg       심볼 (단색, 인쇄용)
• StellaDev-Woven-lockup.svg            가로 락업 (심볼 + StellaDev)
• StellaDev-Woven-lockup-reversed.svg   다크 리버스 락업
• StellaDev-Woven-card.svg              명함 (다크, 한글 이름 포함)

.ai (채워진 다각형 방식 · 모든 뷰어 동일 렌더)
• 동일 구성 5종.

인터레이스 지오메트리
--------------------
두 사각형이 8개 교차점에서만 번갈아 위/아래로 엮입니다. 아래로 지나가는 획을
교차점에서 실제로 끊은 깨끗한 벡터 패스입니다(오버페인트 없음) — 각 획을 그대로
선택·편집할 수 있습니다.

폰트
----
워드마크/본문은 Sora(무료, Google Fonts)를 기준으로 지정했습니다. 없으면 Helvetica로
대체됩니다. 최종 배포 시 Sora 설치 후 사용하거나, Illustrator에서 [윤곽선 만들기].
SVG 명함에는 한글 "하용욱"이 포함되어 있습니다(한글 폰트 필요).
